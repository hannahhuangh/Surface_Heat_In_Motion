Earthquakes in Motion

Exploratory Analysis · DSC 106 · Project 3 Checkpoint

DATASET: USGS EARTHQUAKE CATALOG
FORMAT: GEOJSON
PERIOD: 2025 · GLOBAL · MAG 4.5+

This exploratory analysis uses USGS Earthquake Catalog GeoJSON data for global earthquakes in 2025 with magnitude 4.5 or higher. These charts helped shape the final interactive visualization by identifying useful encodings: circle size for magnitude, color for depth, and a time slider for monthly activity.
